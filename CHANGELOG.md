## [0.1.0] - 2025-11-18

**PR:** #7  

**Breaking Changes**
- 공개 API가 Ollama 기반 이름에서 SUH-AIDER 기반으로 변경되었습니다(서비스, DTO, 예외 포함).

**New Features**
- 클래스 기반 JSON 스키마 자동 생성 기능이 추가되었습니다.@AICLASS, @AiSchema, @AiHidden, @AiArraySchema 주석을 사용하여 스키마를 정의합니다.
- 보안 헤더 이름 및 형식을 커스터마이징할 수 있는 새로운 옵션이 추가되었습니다.
- API 키 설정이 선택사항으로 변경되었습니다.

## [0.0.10] - 2025-11-17

**PR:** #6  

**새로운 기능**
- JSON Schema를 통한 구조화된 응답 형식 지정 기능 추가
- 프롬프트 전처리 및 후처리, 응답 타임아웃 커스터마이징 옵션 추가
- 응답 검증 및 정제 기능 추가

**문서**
- README 전면 개편 (설치, 설정, 사용 예제, API 레퍼런스 포함)
- JSON Schema 사용 가이드 문서 추가
